# Movie Listing App

A movie search application built with React, Vite, and the OMDb API.

## Features

- Search movies, TV shows, and episodes
- View movie details including ratings, cast, and plot
- Filter results by type
- Responsive design for all devices
- Secure API handling using Vercel Serverless Functions

## Tech Stack

- React
- Vite
- OMDb API
- Vercel

## Run Locally

```bash
npm install
npm run dev
```

Create a `.env` file in the project root:

```env
OMDB_API_KEY=your_api_key_here
```

## Deployment

Add `OMDB_API_KEY` to Vercel Environment Variables before deploying.
